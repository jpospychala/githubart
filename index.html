Pimp my Github calendar

Draw the calendar the way you want it to look, turn into commits and push to github!

Click the squares on the calendar below

Importing

To import an existing github calendar, paste the contents of https://github.com/users/USERNAME/contributions_calendar_data in textbox below and press Import.

Saving as your calendar

After playing around with the calendar above, press Export to generate list of git commands that will make your github calendar look like the one above.
